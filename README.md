# Moonicorn Landing Page

Sebuah *landing page* untuk proyek **Moonicorn**, dibangun sebagai portofolio website menggunakan HTML, CSS, dan JavaScript. Tampilan desain simple dan responsif, cocok untuk digunakan sebagai halaman awal proyek atau promosi.

## Fitur

- Layout responsif (mobile & desktop)  
- Hero section dengan gambar / teks menarik  
- Bagian “About” atau deskripsi proyek  
- Seksi fitur / keunggulan  
- Kontak atau CTA (Call To Action)  

## Teknologi

- HTML  
- CSS  
- JavaScript (vanilla)  
- Bootstrap / CSS custom
